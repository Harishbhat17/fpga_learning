# Quartus TCL Scripts

> **Navigation:** [← Quartus README](../README.md)

| Script | Description |
|--------|-------------|
| [timing_closure_flow.tcl](timing_closure_flow.tcl) | Full compile and timing check automation |

## Usage

```bash
# From command line:
quartus_sh --script=scripts/timing_closure_flow.tcl

# From Quartus TCL console:
source scripts/timing_closure_flow.tcl
```
